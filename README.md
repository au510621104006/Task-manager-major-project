# 📝 Task Manager Full Stack Web App

## 🚀 Project Overview

This is a full stack Task Manager web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It provides secure authentication and complete task management features for users.

Users can register, login, and manage their daily tasks efficiently with options like create, update, delete, and mark tasks as completed.

---

## ✨ Features

- 🔐 User Signup & Login (JWT Authentication)
- 🔒 Protected Routes (Authorization)
- ➕ Create Tasks
- 📖 View Tasks
- ✏️ Update Tasks
- ❌ Delete Tasks
- ✅ Toggle Task Completion
- 🔍 Search Tasks
- 📊 Filter & Sort Tasks

---

## 🛠️ Tech Stack

**Frontend:**
- React.js

- React Router DOM

**Backend:**
- Node.js
- Express.js

- Mongoose
